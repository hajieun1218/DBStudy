칸과 골든글로브에 이어 '불모지' 아카데미영화제까지 평정한 봉준호 영화감독 일대기를 다룬 아동용 만화가 나왔다.

13일 아동출판계에 따르면 주니어RHK는 최근 직업 탐구 만화인 '아이엠' 시리즈 최신작으로 봉준호를 주제로 한 '아이엠 봉준호'를 출간했다.

봉 감독의 생애와 활동을 통해 영화감독의 세계를 어린이들에게 알려주는 내용이다.

밤 새워 수많은 영화를 보고서 친구들에게 온종일 이야기할 정도로 영화광이던 초등학생 시절부터 한국영화아카데미에서 실제 영화 제작을 고민하던 모습, 첫 상업 영화를 만들던 순간 등을 보여준다.
